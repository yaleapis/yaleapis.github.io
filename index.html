Courses

CourseTable API

Use the CourseTable API to get course information in a clean format. The API is undocumented, but to access all the data, log into CourseTable and then visit https://api.coursetable.com/api/static/catalogs/202203.json to access the JSON file. Further information coming soon.

Courses (Official, But Worse)

Yale Courses and CourseSections API

The Courses and CourseSections API can be used to fetch course data as well, but it is far more awkward to use. the CourseTable API to get course information in a clean format. The API is undocumented, but to access all the data, log into CourseTable and then visit https://api.coursetable.com/api/static/catalogs/202203.json to access the JSON file.
